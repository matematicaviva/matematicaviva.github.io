01 — Definizione

Le Equazioni

Cos'è un'equazione?

Un'equazione è un'uguaglianza tra due espressioni letterali per la quale cerchiamo gli eventuali valori che, sostituiti a una o più lettere, dette incognite, la rendono vera.

ax + b  =  cx + d    (a, b, c, d ∈ ℝ)

Chiamiamo primo membro l'espressione a sinistra del segno =, secondo membro quella a destra.

Analisi di: 3x + 5 = 2x − 1

=
2x − 1

Radici o soluzioni

In un'equazione chiamiamo radici o soluzioni i valori, da attribuire alle incognite, che rendono uguali il primo e il secondo membro. Risolvere un'equazione vuol dire trovarne tutte le soluzioni!

Equazione in forma normale

Un'equazione è in forma normale se è del tipo P(x)=0, con P(x) polinomio ridotto a forma normale. Il grado di P(x) è il grado dell'equazione!

02 — Equivalenza

I Principi di equivalenza

Due equazioni nelle stesse incognite sono equivalenti se hanno le stesse soluzioni.

⚖️ Primo e secondo principio

1° Principio

Aggiungendo o sottraendo a entrambi i membri di un'equazione uno stesso numero, o espressione letterale, otteniamo un'equazione equivalente a quella data.

x + 3 = 7
x + 3 − 3 = 7 − 3
x = 4 ✓
2° Principio

Moltiplicando o dividendo entrambi i membri di un'equazione per uno stesso numero o espressione letterale diversi da zero, otteniamo un'equazione equivalente a quella data.

2x = 10
2x ÷ 2 = 10 ÷ 2
x = 5 ✓

Puoi pensarla come una bilancia in equilibrio: qualunque operazione fai su un piatto, devi farla anche sull'altro — questo è il principio alla base dei principi di equivalenza.

Regola del trasporto

Data un'equazione, ne otteniamo una equivalente se trasportiamo un termine da un membro all'altro cambiandogli segno.

Regola di cancellazione

Data un'equazione, ne otteniamo una equivalente se in entrambi i membri cancelliamo termini uguali.

Regola del cambiamento di segno

Data un'equazione, ne otteniamo una equivalente se cambiamo segno a tutti i suoi termini.

03 — Primo tipo

Equazione lineare o di primo grado Determinata

Supponiamo che l'equazione sia nella forma ax=b con a ≠ 0, allora possiamo dividere entrambi i membri per "a":

ax = b   con a ≠ 0   ->   x = b/a

Esiste esattamente un valore di x che soddisfa l'equazione. L'insieme soluzione S ha un solo elemento. L'equazione data è detta determinata

✅ Esempio 1
4x − 3 = 2x + 9
4x − 2x = 9 + 3
2x = 12
x = 6
✅ Esempio 2
5x − 10 = 0
5x = 10
x = 2

📝 Esempio Guidato

Risolvere: 4x − 3 = 2x + 9

1. Spostiamo x a sinistra: 4x − 2x = 9 + 3
2. Semplifichiamo: 2x = 12
3. Dividiamo per 2: x = 6
4. Verifica: 4·6−3 = 21 = 2·6+9 ✓  DETERMINATA

04 — Secondo tipo

Equazione Indeterminata

L'equazione ax=b è indeterminata se a=0 e b=0, quindi l'equazione è nella forma 0X=0:

0 · x = 0   →   0 = 0   (sempre vero!)

L'uguaglianza è vera per qualunque valore di x: l'equazione ha infinite soluzioni (S = ℝ).

♾️ Esempio 1
4(x + 1) = 4x + 4
4x + 4 = 4x + 4
0 = 0
S = ℝ
♾️ Esempio 2
3(x − 2) = 3x − 6
3x − 6 = 3x − 6
0 = 0
S = ℝ

📝 Esempio Guidato

Risolvere: 3(x + 2) = 3x + 6

1. Espandiamo: 3x + 6 = 3x + 6
2. Raccogliamo: 3x − 3x = 6 − 6 → 0 = 0
3. 0 = 0 è vera per ogni valore di x!
4. INDETERMINATA — S = ℝ (infinite soluzioni)

05 — Terzo tipo

Equazione Impossibile

L'equazione ax=b è impossibile se a=0 e b ≠ 0:

0 · x = b   con b ≠ 0   →   0 = b   (assurdo!)

Nessun valore di x renderà mai vera un'uguaglianza come "0 = 5". L'equazione non ha soluzioni (S = ∅).

🚫 Esempio 1
x + 3 = x + 8
x − x = 8 − 3
0 = 5 ✗ → S = ∅
🚫 Esempio 2
2(x + 1) = 2x + 5
2x + 2 = 2x + 5
2 = 5 ✗ → S = ∅

📝 Esempio Guidato

Risolvere: 5x + 7 = 5x − 2

1. Spostiamo x: 5x − 5x = −2 − 7
2. Risultato: 0 = −9 — uguaglianza FALSA!
3. 0 = −9 è falsa per qualunque x.
4. IMPOSSIBILE — S = ∅ (nessuna soluzione)

06 — Caso speciale

Le Equazioni Fratte

⚠️ Quando l'incognita è al denominatore

Un'equazione è fratta quando l'incognita compare in uno o più denominatori. Il denominatore non può essere zero, quindi dobbiamo tenere conto delle condizioni di esistenza delle frazioni algebriche presenti!

Esempio fratta

1/(x−2) + 3/x = 5/6

Denominatori: (x−2), x, 6

C.E. — Condizioni di Esistenza

x − 2 ≠ 0 → x ≠ 2
x ≠ 0

⚡ Prima cosa da fare sempre!

📋 Metodo in 5 passi

  1. Trova le C.E.: poni ogni denominatore ≠ 0 e ricava i valori esclusi.
  2. Calcola il M.C.D. di tutti i denominatori.
  3. Esegui le operazioni fra le frazioni algebriche presenti nell'equazione
  4. Moltiplica tutto per il M.C.D.: i denominatori scompaiono.
  5. Risolvi l'equazione intera risultante.
  6. Verifica con le C.E.! Se la soluzione è un valore escluso → S = ∅.

📝 Esempio Guidato

Risolvere: 2/(x−1) = 3/(x+2)

1. C.E.: x ≠ 1 e x ≠ −2
2. M.C.D. = (x−1)(x+2). Eseguo le somme fra frazioni algebriche e poi moltiplico per il M.C.D.: 2(x+2) = 3(x−1)
3. Espando e risolvo: 2x+4 = 3x−3 → x = 7
4. x=7 ≠ 1 e x=7 ≠ −2 ✓  ACCETTATA → x = 7

🏋️ Esercizio
