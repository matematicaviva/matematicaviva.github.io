Disequazioni di secondo grado e parabola

01 — Cos'è una disequazione di secondo grado?

Una disequazione di secondo grado è una disuguaglianza del tipo:

📐 ax² + bx + c > 0   |   < 0   |   ≥ 0   |   ≤ 0

Il nostro obiettivo è trovare tutti i valori di x per cui la disuguaglianza è vera. Non un singolo numero, ma un insieme (spesso un intervallo) di valori.

💡 Differenza con le equazioni di 2° grado
L'equazione ax² + bx + c = 0 cerca i valori puntuali che azzerano l'espressione: al massimo 2 numeri.
La disequazione cerca tutti i valori per cui l'espressione è positiva (o negativa): la risposta è di solito un intervallo o un'unione di intervalli.

Il metodo risolutivo si basa su un'idea geometrica molto semplice, che vedremo subito nella sezione successiva dedicata alla parabola.

02 — La Parabola: cos'è e come si usa

Ad ogni espressione ax² + bx + c possiamo associare una curva nel piano chiamata parabola:

📈 y = ax² + bx + c

La parabola è una curva simmetrica con la concavità rivolta verso l'alto "U" o verso il basso "∩", a seconda del segno del coefficiente a. Ecco i due casi:

🟢 a > 0 — Concavità verso l'alto (U)
🔴 a < 0 — Concavità verso il basso (∩)
⚠️ Il segno di a è fondamentale
Tutto il metodo risolutivo dipende dal sapere se la parabola è "a U" o "a ∩". Controlla sempre il segno di a prima di procedere.
03 — Il Discriminante Δ: quante volte la parabola interseca l'asse x?

Prima di risolvere la disequazione dobbiamo sapere come la parabola si posiziona rispetto all'asse x. Questa informazione è contenuta nel discriminante:

🔢 Δ = b² − 4ac

Il discriminante ci dice quante soluzioni reali ha l'equazione associata ax² + bx + c = 0, cioè quante volte la parabola interseca l'asse x:

Segno di ΔGeometriaSignificato
Δ > 0Due intersezioni distinte: x₁ e x₂ (x₁ < x₂)La parabola interseca l'asse x in due punti distinti
Δ = 0Un solo punto di tangenza: x₀La parabola interseca l'asse x in un unico punto (due punti coincidenti)
Δ < 0Nessuna intersezioneLa parabola è completamente sopra o completamente sotto l'asse x
📝 Esempio di calcolo di Δ
Data: 2x² − 3x + 1 > 0 — qui a = 2, b = −3, c = 1
1️⃣ Δ = b² − 4ac = (−3)² − 4 · 2 · 1 = 9 − 8 = 1 > 0 → la parabola interseca l'asse in due punti distinti ✅
04 — La Procedura in 4 Passi

Ogni volta che incontri una disequazione di secondo grado, segui sempre questi 4 passi nell'ordine:

  1. Porta tutto al primo membro (forma canonica)
    Riordina in modo da avere ax² + bx + c > 0 (<0) con a ≠ 0. Se necessario, moltiplica per −1 (ricorda di invertire il verso della disequazione!)
  2. Calcola il discriminante Δ = b² − 4ac
    Questo numero decide quante radici reali ha l'equazione associata, cioè quante volte la parabola interseca l'asse x.
  3. Se Δ ≥ 0, calcola le radici x₁ e x₂
    Con la formula: x = (−b ± √Δ) / 2a. Chiama x₁ la radice minore e x₂ quella maggiore.
  4. Scrivi la soluzione in base al segno di Δ e di a
    Il segno di a (concavità della parabola) è fondamentale: determina se la parabola è del tipo U o del tipo ∩, e quindi dove si trova rispetto all'asse x.
📊 I casi concreti nelle sezioni successive
A seconda del valore di Δ si presentano tre situazioni distinte, ciascuna con le proprie regole di soluzione:
• Δ > 0 → l'equazione associata ha due radici reali distinte x₁ e x₂
• Δ = 0 → l'equazione associata ha due radici reali e coincidenti x₁=x₂=x₀
• Δ < 0 → l'equazione associata non ha radici reali
Nelle sezioni successive vedremo esattamente come ricavare la soluzione della disequazione in ciascuno di questi tre casi.

06 — Caso 1: Δ > 0

🧮 Risoluzione Algebrica

Quando Δ > 0, l'equazione associata ha due radici reali distinte x₁ e x₂ (con x₁ < x₂). Il trinomio si scompone come:

🔍 ax² + bx + c = a(x − x₁)(x − x₂)

Per determinare gli intervalli soluzione confrontiamo il verso della disequazione con il segno di a e applichiamo la regola del DICE:

🎲 DI — Discordi → Interni
Se il verso della disequazione è opposto al segno di a → soluzione interna: x₁ < x < x₂
🎲 CE — Concordi → Esterni
Se il verso della disequazione è uguale al segno di a → soluzione esterna: x < x₁ ∨ x > x₂
📝 Esempio Guidato Passo per Passo
Risolvere: x² − 5x + 6 > 0
1️⃣ Identifica i coefficienti: a = 1, b = −5, c = 6
2️⃣ Calcola Δ: Δ = (−5)² − 4·1·6 = 25 − 24 = 1 > 0 → due soluzioni reali e distinte ✅
3️⃣ Calcola le radici: x₁ = (5 − √1) / 2 = 2   |   x₂ = (5 + √1) / 2 = 3
4️⃣ Applica DICE: a = 1 > 0, verso disequazione > 0 → segni concordi → Esterni (CE)
5️⃣ Soluzione: x < 2 ∨ x > 3

📈 Risoluzione Grafica

🔍 L'intuizione geometrica centrale
Per risolvere ax² + bx + c > 0 bisogna rispondere alla domanda: "Per quali valori di x la parabola sta sopra l'asse x?"
Per risolvere ax² + bx + c < 0 bisogna rispondere alla domanda: "Per quali valori di x la parabola sta sotto l'asse x?"
Le soluzioni sono le ascisse dei punti della parabola che stanno sopra o sotto l'asse x.
Tutto il metodo si riduce a leggere il grafico.

Clicca sui pulsanti per cambiare il verso della disequazione e osserva come cambia la soluzione (in azzurro sull'asse):

Segno richiesto: > 0 | Soluzione: x < 1 ∨ x > 2
⚠️ Attenzione ai segni ≤ e ≥
Con disuguaglianza stretta (< o >) i punti x₁ e x₂ sono esclusi dalla soluzione (cerchietti aperti sul grafico).
Con disuguaglianza larga (≤ o ≥) i punti x₁ e x₂ sono inclusi (cerchietti chiusi).
07 — Caso 2: Δ = 0

🧮 Risoluzione Algebrica

Quando Δ = 0, l'equazione ha due soluzioni reali e coincidenti: x₀ = −b/(2a). Il trinomio è il prodotto di a per un quadrato perfetto:

🔍 ax² + bx + c = a(x − x₀)²

Poiché (x − x₀)² ≥ 0 sempre, il segno dell'espressione dipende solo da a:

ΔSegno di a> 0≥ 0< 0≤ 0
Δ = 0a > 0ℝ \ {x₀}∀x ∈ ℝ∅{x₀}
Δ = 0a < 0∅ {x₀} ℝ \ {x₀}∀x ∈ ℝ
📝 Esempio Svolto
Risolvere: x² − 6x + 9 ≤ 0
1️⃣ Coefficienti: a = 1, b = −6, c = 9
2️⃣ Δ: Δ = (−6)² − 4·1·9 = 36 − 36 = 0 → due soluzioni reali e coincidenti
3️⃣ Radice: x₀ = −(−6)/(2·1) = 3
4️⃣ Interpretazione: a = 1 > 0, quindi (x−3)² ≥ 0 sempre → mai < 0
5️⃣ Chiediamo ≤ 0: vale solo dove l'espressione è uguale a 0, cioè x = 3

📈 Risoluzione Grafica

La parabola interseca l'asse x in un solo punto. Osserva che non c'è mai una regione "tra due radici":

Segno richiesto: > 0 | Soluzione: ∀x ≠ 1
📌 Nota Bene
Un quadrato è sempre ≥ 0. Moltiplicato per a > 0 rimane ≥ 0. L'unico momento in cui vale 0 è quando x = x₀.
Quindi non può mai essere strettamente negativo (con a > 0).
08 — Caso 3: Δ < 0

🧮 Risoluzione Algebrica

Quando Δ < 0, l'equazione non ha soluzioni reali. La parabola è completamente sopra o completamente sotto l'asse x — non lo interseca mai.

🟢 Se a > 0:
ax² + bx + c > 0 → ∀x ∈ ℝ  |  ax² + bx + c < 0 → ∅
🔴 Se a < 0:
ax² + bx + c < 0 → ∀x ∈ ℝ  |  ax² + bx + c > 0 → ∅
📝 Esempio Guidato
Risolvere: x² + 2x + 5 > 0
1️⃣ Coefficienti: a = 1, b = 2, c = 5
2️⃣ Δ: Δ = 4 − 20 = −16 < 0 → nessuna radice reale
3️⃣ Segno di a: a = 1 > 0
4️⃣ Chiediamo > 0 → Soluzione: ∀x ∈ ℝ

📈 Risoluzione Grafica

Nota come la parabola non interseca mai l'asse x. Cambia il segno di a per vedere l'effetto opposto:

Verso disequazione: > 0 | Soluzione: ∀x ∈ ℝ
🔍 Logica intuitiva
Se la parabola è sempre sopra l'asse (non lo interseca mai), allora y > 0 per tutti i valori di x. La soluzione è l'intero ℝ.
Viceversa, la stessa parabola non può mai stare sotto l'asse: nessun x verifica y < 0 → soluzione vuota ∅.
⚠️ Errore comune
Molti studenti dimenticano di considerare il segno di a nel caso Δ < 0. Con a < 0 è tutto capovolto rispetto a a > 0!
09 — Visione d'insieme: Algebra e Grafico

I due approcci portano sempre allo stesso risultato. Ecco come si corrispondono:

🧮 Vista Algebrica
Δ > 0 → a(x−x₁)(x−x₂) > 0 (<0)
Δ = 0 → a(x−x₀)² > 0 (<0)
Δ < 0 → Segno fisso = segno di a
📊 Vista Grafica
Δ > 0 → Interseca l'asse in 2 punti distinti
Δ = 0 → Tangente: interseca l'asse in 2 punti coincidenti
Δ < 0 → Completamente sopra o sotto l'asse

11 — Esercizi Guidati

Prova a risolvere ciascun esercizio. Usa il pulsante "Suggerimento" se sei bloccato, poi "Soluzione" per vedere i passi completi.

1. Risolvi: x² − x − 6 < 0

💡 Suggerimento
Identifica a, b, c. Calcola Δ. Trova le radici. Applica DICE controllando il segno di a e il verso della disequazione.
✅ Soluzione
➊ Coefficienti: a = 1, b = −1, c = −6
➋ Calcola Δ: Δ = (−1)² − 4·1·(−6) = 1 + 24 = 25 > 0 → due soluzioni reali e distinte
➌ Soluzioni: x₁ = (1 − 5)/2 = −2 e x₂ = (1 + 5)/2 = 3
➍ DICE: a = 1 > 0, verso disequazione < 0 → Discordi → Interni

✅ Soluzione: −2 < x < 3

2. Risolvi: −x² + 2x − 1 < 0

💡 Suggerimento
Attenzione: a è negativo! Calcola Δ. Se Δ = 0, ricorda che il segno dipende solo da a e dal quadrato perfetto.
✅ Soluzione
➊ Coefficienti: a = −1, b = 2, c = −1 (attenzione: a è negativo!)
➋ Calcola Δ: Δ = 4 − 4·(−1)·(−1) = 4 − 4 = 0 → due soluzioni reali e coincidenti
➌ Soluzioni: x₀ = −2 / (2·(−1)) = 1
➍ Δ = 0 e a = −1 < 0: la parabola è sempre ≤ 0. Verso disequazione < 0 → vero per tutti x ≠ 1

✅ Soluzione: ∀x ≠ 1 (cioè x ∈ ℝ \ {1})

3. Risolvi: x² + x + 1 > 0

💡 Suggerimento
Calcola Δ. Se è negativo, controlla solo il segno di a.
✅ Soluzione
➊ Coefficienti: a = 1, b = 1, c = 1
➋ Calcola Δ: Δ = 1 − 4 = −3 < 0 → nessuna radice reale
➌ Δ < 0 e a = 1 > 0: la parabola è sempre sopra l'asse
➍ Chiediamo > 0: è sempre verificata → soluzione tutto ℝ

✅ Soluzione: ∀x ∈ ℝ

4. (Sfida) Risolvi: 2x² − 7x + 3 ≥ 0

💡 Suggerimento
Porta in forma canonica. Calcola Δ. Trova le radici. Applica DICE con ≥ (estremi inclusi).
✅ Soluzione
➊ Coefficienti: a = 2, b = −7, c = 3
➋ Δ = (−7)² − 4·2·3 = 49 − 24 = 25 > 0 → due soluzioni reali e distinte
➌ Soluzioni: x₁ = (7 − 5)/4 = 1/2 e x₂ = (7 + 5)/4 = 3
➍ DICE: a = 2 > 0, verso disequazione ≥ 0 → Concordi → Esterni (con estremi inclusi perché ≥)

✅ Soluzione: x ≤ 1/2 ∨ x ≥ 3
