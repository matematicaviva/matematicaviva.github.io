01 — Definizione

Disequazioni di Secondo Grado

Cos'è una disequazione di secondo grado?

Una disequazione di secondo grado è una disuguaglianza in cui compare un'incognita elevata al quadrato. Cerchiamo i valori di x che rendono vera la disuguaglianza.

ax² + bx + c  > 0  oppure  < 0  oppure  ≥ 0  oppure  ≤ 0

La risoluzione è strettamente legata alla parabola y = ax² + bx + c: le soluzioni della disequazione corrispondono agli intervalli in cui la parabola è sopra (o sotto) l'asse x.

La chiave di tutto è il discriminante Δ, che determina quante intersezioni ha la parabola con l'asse x — e quindi quante soluzioni ha l'equazione associata.

02 — Strumento chiave

Il Discriminante

Δ = b² − 4ac

Il discriminante ci dice quante soluzioni reali ha l'equazione associata ax² + bx + c = 0, e quindi quante volte la parabola interseca l'asse x.

Δ > 0

Due soluzioni reali distinte

La parabola taglia l'asse x in 2 punti

Δ = 0

Una soluzione reale doppia

La parabola è tangente all'asse x

Δ < 0

Nessuna soluzione reale

La parabola non tocca l'asse x

03 — Primo caso

Caso 1: Δ > 0

Quando Δ > 0, l'equazione associata ha due soluzioni reali distinte x₁ e x₂. Il trinomio si può scomporre:

a(x − x₁)(x − x₂) > 0   (oppure < 0)

Per determinare gli intervalli soluzione, applichiamo la regola del DICE:

🎲 DI — Discordi → Interni

Se il segno del prodotto è diverso da quello di a, la soluzione è l'intervallo interno tra x₁ e x₂.

🎲 CE — Concordi → Esterni

Se il segno del prodotto è uguale a quello di a, la soluzione è l'unione degli intervalli esterni.

Visualizzazione Interattiva

📝 Esempio Guidato

Risolvere: x² − 5x + 6 > 0

1. Calcoliamo Δ = 25 − 24 = 1 > 0 → due soluzioni
2. x₁ = (5−1)/2 = 2, x₂ = (5+1)/2 = 3
3. a = 1 > 0 e chiediamo > 0 → segni concordi → soluzione esterna
4. Soluzione: x < 2 ∨ x > 3

04 — Secondo caso

Caso 2: Δ = 0

Quando Δ = 0, l'equazione ha una soluzione reale doppia x₀ = −b/(2a). La parabola è tangente all'asse x.

ax² + bx + c > 0

Verificata ∀x ≠ x₀ (se a > 0)

Sol: x ∈ ℝ \ {x₀}

ax² + bx + c < 0

Mai verificata (se a > 0)

Sol: ∅

Con ≥ e ≤ il punto x₀ viene incluso o meno nella soluzione.

📝 Esempio Svolto

x² − 6x + 9 ≤ 0

  1. Δ = 36 − 36 = 0 → soluzione doppia
  2. x₀ = 6/2 = 3
  3. a = 1 > 0 → parabola rivolta verso l'alto, sempre ≥ 0
  4. Chiediamo ≤ 0: la parabola tocca zero solo in x₀
  5. Soluzione: x = 3

05 — Terzo caso

Caso 3: Δ < 0

Quando Δ < 0, l'equazione associata non ha soluzioni reali. La parabola è completamente sopra o sotto l'asse x.

Se a > 0

Parabola tutta sopra l'asse x

> 0 → ∀x ∈ ℝ  |  < 0 → ∅

Se a < 0

Parabola tutta sotto l'asse x

< 0 → ∀x ∈ ℝ  |  > 0 → ∅

Cambia il segno di a per vedere come cambia il risultato:

📝 Esempio Guidato

x² + 2x + 5 > 0

  1. Δ = 4 − 20 = −16 < 0 → nessuna soluzione reale
  2. a = 1 > 0 → parabola tutta sopra l'asse x
  3. Chiediamo > 0: sempre verificata
  4. Soluzione: ∀x ∈ ℝ

06 — Metodo mnemonico

La Regola del DICE

La regola del DICE è un metodo mnemonico per ricordare rapidamente se la soluzione è interna o esterna rispetto alle radici x₁ e x₂ (con x₁ < x₂).

🎲

DI

Discordi → Interni

Il segno richiesto è opposto al segno di a

x₁ < x < x₂

Es: a > 0 e chiediamo < 0

🎲

CE

Concordi → Esterni

Il segno richiesto è uguale al segno di a

x < x₁ ∨ x > x₂

Es: a > 0 e chiediamo > 0
