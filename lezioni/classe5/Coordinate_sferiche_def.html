Cosa sono le Coordinate Sferiche?

Un sistema alternativo per localizzare un punto nello spazio 3D usando una distanza e due angoli.

r Distanza radiale

Distanza dall'origine al punto P. Sempre ≥ 0.

θ (theta) Angolo azimutale

Angolo dal verso positivo di x verso la proiezione di r nel piano xy. Intervallo: [0, 2π).

φ (phi) Angolo polare

Angolo rispetto all'asse z positivo. Intervallo: [0, π].

🖱️ Trascina per ruotare — asse z verso l'alto

Relazione con le Coordinate Cartesiane

Sferiche → Cartesiane

x = r sinφ cosθ
y = r sinφ sinθ
z = r cosφ

Cartesiane → Sferiche

r = √(x² + y² + z²)
θ: cosθ = x/ρ, sinθ = y/ρ
φ = arccos(z / r)

📐 Come determinare il quadrante corretto di θ

La proiezione del vettore r sul piano xy ha modulo ρ = r sin φ. L'angolo θ è quello che questa proiezione forma con l'asse x positivo. Per identificare il quadrante corretto si usano entrambe le componenti:

cos θ = x / (r sin φ) = x / ρ
sin θ = y / (r sin φ) = y / ρ

Il segno di cos θ e sin θ indica il quadrante:

I quadrante
x > 0, y > 0
cos θ > 0, sin θ > 0
θ ∈ (0, π/2)
II quadrante
x < 0, y > 0
cos θ < 0, sin θ > 0
θ ∈ (π/2, π)
III quadrante
x < 0, y < 0
cos θ < 0, sin θ < 0
θ ∈ (π, 3π/2)
IV quadrante
x > 0, y < 0
cos θ > 0, sin θ < 0
θ ∈ (3π/2, 2π)

💡 Procedura: calcola ρ = r sin φ, poi ricava cos θ = x/ρ e sin θ = y/ρ. I segni di entrambi determinano univocamente il quadrante, e quindi il valore esatto di θ ∈ [0, 2π).

Applicazioni Pratiche

⚡

Fisica

Campi elettrici e gravitazionali con simmetria sferica. Il potenziale coulombiano V = kq/r si semplifica notevolmente.

Astronomia

Posizione dei corpi celesti: ascensione retta e declinazione sono coordinate sferiche sulla sfera celeste.

Momento Interattivo 🧠

Domanda 1: Un punto ha coordinate cartesiane P = (0, 0, 3). Quanto vale φ?

Domanda 2: Se P = (1, 1, √2), quanto vale r?

Esercizio Guidato: Converti P = (1, √3, 2) in sferiche

Mostra soluzione
r = √(1 + 3 + 4) = √8 = 2√2 ≈ 2.83
θ: ρ = r sin φ = 2√2 · sin(π/4) = 2√2 · (√2/2) = 2 → cos θ = x/ρ = 1/2 > 0, sin θ = y/ρ = √3/2 > 0 → I quadrante → θ = arccos(1/2) = π/3 ≈ 1.047 rad
φ = arccos(z/r) = arccos(2/2√2) = arccos(1/√2) = π/4 ≈ 0.785 rad
